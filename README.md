# Energy Consumption and Trend Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive **Energy Consumption Dashboard** built using Power BI.
The dashboard analyzes energy usage across different **cities, buildings, and consumption types** such as electricity, gas, and water.

The aim of this project is to demonstrate how **data visualization tools help transform raw data into meaningful insights** for better decision-making.

## Objectives

* Analyze energy consumption trends over time
* Identify the highest energy-consuming cities and buildings
* Compare electricity, gas, and water consumption
* Build an interactive dashboard using Power BI
* Develop practical skills in data visualization and analytics

## Tools & Technologies Used

* Microsoft Power BI
* Power Query (Data Transformation)
* DAX (Data Analysis Expressions)
* Excel / CSV Dataset

## Dashboard Features

* **KPI Cards** displaying total cost and units consumed
* **Line Chart** showing energy consumption trends over time
* **Bar Chart** comparing energy usage across buildings
* **Donut Chart** displaying distribution of energy types
* **City-wise analysis** of energy consumption
* **Interactive slicers** for filtering by city, building, and energy type

## Key Insights

* Electricity accounts for the **largest share of energy consumption**.
* Some cities show **higher energy usage trends compared to others**.
* Certain buildings demonstrate **irregular electricity consumption patterns**.
* Visualization helps identify **energy usage patterns and optimization opportunities**.

## How to Use

1. Download the `.pbix` file from the repository.
2. Open it using **Power BI Desktop**.
3. Use the filters and slicers to explore the dashboard interactively.
